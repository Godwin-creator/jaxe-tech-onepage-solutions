# JAXE TECH – Agence digitale (Togo)

## Choix de design

- **Palette professionnelle** : bleu profond (`#0B2B4A`) pour la confiance & la technologie ; orange doré (`#F39C12`) pour les actions (conversion) ; gris neutres pour la lisibilité.
- **Mode clair / sombre** : toggle avec mémorisation – accessibilité et confort visuel.
- **Typographie** : *Cormorant Garamond* (titres, élégance) + *DM Sans* (corps, moderne).
- **Animations** : loader d’entrée, révélations au scroll, effets de survol – navigation fluide.
- **Responsive** : menu hamburger, grilles adaptatives, padding ajusté (mobile/tablette/desktop).

## Approche marketing (conversion → clients)

- **CTA stratégiques** : « Demander un devis » et « Nous contacter » en hero + bouton flottant dans la nav.
- **Preuve sociale** : blocs de statistiques (`+50 projets`, `100% satisfaction`, `24/7 support`) pour rassurer.
- **Formulaire optimisé** : champs limités (nom, email, message), validation en temps réel, feedback immédiat.
- **Hiérarchie visuelle** : cartes services et avantages pour un scan rapide ; badge « depuis 2020 » renforce la crédibilité.

## Optimisations SEO (ciblage Togo)

- **Balises méta** : description et keywords incluant « agence digitale Togo », « création site web Lomé », « SEO Togo ».
- **Structure sémantique** : une seule `<h1>`, des `<h2>` par section, des `<h3>` pour les sous-titres.
- **Contenu local** : adresse (Avédjii, Lomé), téléphone (`+228 97 99 99 43`), horaires dans le footer.
- **Accessibilité** : attributs `alt` sur les images (ou équivalents textuels), contrastes WCAG valides.
- **Performances** : code léger, pas de dépendances lourdes, chargement rapide.
